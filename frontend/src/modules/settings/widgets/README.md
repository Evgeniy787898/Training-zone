# Settings Widgets

Виджеты для декомпозиции SettingsPage.vue согласно ARCHITECTURE_IMPROVEMENT_PLAN.md (SETT-R01 - SETT-R06).

## Планируемые виджеты:
- ProfileWidget.vue (SETT-R02) — текущая программа, кнопка смены
- NotificationsWidget.vue (SETT-R03) — toggle уведомлений + time picker
- SystemStatusWidget.vue (SETT-R04) — health check, метрики
- PinWidget.vue (SETT-R05) — управление PIN-кодом
- ThemeWidget.vue (SETT-R06) — выбор темы, акцентный цвет
